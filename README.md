# Sales Data Analysis Project

## Project Overview
This project analyzes an e-commerce sales dataset using Python.
The goal is to understand sales trends, best selling categories, and revenue insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
File used:
ecommerce_sales_data.csv

## Project Steps
1. Load the dataset
2. Explore the data
3. Handle missing values
4. Perform sales analysis
5. Visualize monthly sales

## Key Insights
- Calculated total revenue from the dataset
- Identified the best selling category
- Found top revenue generating category
- Analyzed monthly sales trends

## Output Example
Some important results from the analysis:

Total Revenue: 27123544.51999996

Best Selling Category: Home Decor
